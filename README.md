## Deploying

This site is built with Jekyll and hosted on GitHub Pages. To enable:

1. Go to repository Settings > Pages
2. Under "Source", select "Deploy from a branch"
3. Select the `main` branch and `/ (root)` folder
4. Click Save

The site will be available at `https://[username].github.io/baks_thesis-seminar/`
