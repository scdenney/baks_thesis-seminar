# BA-Eindwerkstuk Seminar

Course website for the BA-Eindwerkstuk Seminar (5723VKO1Y) in the Korean Studies program at Leiden University, AY2025-2026 (Spring).

## Course Information

- **Instructor:** Dr. Steven Denney
- **Meeting:** Fridays, 09:15-11:00, Lipsius 2.05

## Site Contents

- **Home** - Course overview and schedule
- **Syllabus** - Policies and requirements
- **Presentations** - Lecture slides (Weeks 1-4)
- **Assignments** - Assignment details and rubrics
- **Style Guide** - Formatting and citation guidelines
- **Protocol** - Thesis expectations and assessment criteria

## Deploying

This site uses Jekyll with the Cayman theme and custom styling. To enable GitHub Pages:

1. Go to repository Settings > Pages
2. Under "Source", select "Deploy from a branch"
3. Select the `main` branch and `/ (root)` folder
4. Click Save

The site will be available at `https://scdenney.github.io/baks_thesis-seminar/`
